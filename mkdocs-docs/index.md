# OneManCompany Documentation

Welcome to the OneManCompany documentation. OneManCompany is an open-source AI operating system that lets anyone build and run a complete AI-powered company from their browser.

## What is OneManCompany?

You are the CEO — the only human. Everyone else — HR, COO, engineers, designers — are AI employees that think, collaborate, and deliver real work autonomously.

- **Not a chatbot** — a company with org structure, hiring, task management, performance reviews, and knowledge management
- **Not a demo** — delivers production-grade output (games, comics, apps)
- **Not a framework** — a complete platform you can run from your browser
