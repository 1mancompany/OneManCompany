# Canvas Design & Game UI

Visual design fundamentals applied to game interfaces, HUD elements, and in-game graphics.

## Design Principles for Games
- **Visual hierarchy**: guide player attention to critical info (health, objectives)
- **Readability**: ensure UI is legible across screen sizes and resolutions
- **Consistency**: unified color palette, iconography, typography
- **Minimal intrusion**: HUD should inform without blocking gameplay

## Game UI Patterns
- Health bars, mana bars, status indicators
- Inventory grids with drag-and-drop
- Minimap and compass navigation
- Dialog boxes and quest trackers
- Shop/store interfaces with currency display
- Leaderboards and scoreboard overlays
- Settings menus with sliders and toggles
- Loading screens with progress indicators

## Mobile-First Design (Roblox)
- Touch-friendly button sizes (min 44x44pt)
- Thumb-zone placement for controls
- Responsive scaling across devices
- Portrait and landscape adaptation

## Visual Effects
- Tween animations for smooth UI transitions
- Color grading and post-processing aesthetics
- Particle effects for feedback (damage numbers, pickups)
- Screen shake, flash, and juice effects
