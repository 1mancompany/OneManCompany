# Algorithmic Art & Procedural Generation

Expertise in procedural content generation for games using algorithmic art techniques.

## Techniques
- **Noise functions**: Perlin noise, simplex noise for terrain, clouds, textures
- **L-systems**: procedural trees, plants, organic structures
- **Wave Function Collapse**: tile-based procedural level generation
- **Voronoi diagrams**: biome boundaries, crystal patterns, cell structures
- **Fractals**: coastlines, mountains, lightning, recursive decorations

## Game Applications
- Procedural terrain generation (heightmaps, biomes)
- Random dungeon/level layout generation
- Particle systems and visual effects
- Procedural texture and material generation
- NPC appearance/personality randomization
- Loot table and item stat generation
- Weather and environmental effects

## Implementation Patterns
- Seeded randomness for reproducible worlds
- Chunk-based generation for infinite worlds
- LOD-aware generation (less detail at distance)
- Async generation to avoid frame drops
- Caching generated content for performance
