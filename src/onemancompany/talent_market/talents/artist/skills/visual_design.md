# Visual Design

Proficiency in creating visual art, illustrations, and design assets using AI image generation.

## Capabilities
- Generate illustrations and concept art from text descriptions
- Create UI/UX design mockups and visual assets
- Produce marketing materials, posters, and promotional graphics
- Design pixel art, icons, and sprite assets
- Iterate on visual concepts based on creative feedback

## Best Practices
- Always clarify the creative brief before generating artwork
- Consider the target audience and use case when designing
- Maintain consistent visual style across related assets
- Provide multiple variations when exploring creative directions
- Balance aesthetics with functional requirements
