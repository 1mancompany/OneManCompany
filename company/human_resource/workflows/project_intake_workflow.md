# Project Intake Workflow

1. **CEO Request**: CEO submits a new project request.
2. **EA Review**: Executive Assistant (EA) reviews the request and clarifies requirements if necessary.
3. **Task Translation & Breakdown**: EA translates the macro directive into a standard **"One-Page Task Template"**.
   - Breaks down the task into actionable components.
   - Defines measurable acceptance criteria (2-5 points).
   - Sets the estimated budget based on complexity ($0.01 / $0.05 / $0.15+).
4. **Workspace Preparation**:
   - EA initializes the project workspace using the **Version Control and Asset Manager**.
   - Sets up the `changelog.md` to track phase deliverables and changes.
5. **Routing & Dispatch**:
   - EA dispatches tasks to the appropriate responsible officer (COO/CSO/HR) or specific employees.
   - For multi-domain tasks, EA splits and dispatches each part separately.
   - **快速通道（Fast Track）**：仅涉及单一系统工具操作且直接面向CEO的简单任务（如起草邮件、创建日历事件、查询信息），由EA直接执行，无需经过COO/HR等中间节点流转。
6. **Execution**: Responsible officer coordinates execution within the assigned workspace.
7. **Acceptance & Review**:
   - Responsible officer reviews against acceptance criteria (`accept_project`).
   - EA performs final quality gate review (`ea_review_project`).
