# Web Research

Ability to search the web for technical documentation, code examples, and solutions.

## Capabilities
- Search for API documentation and library references
- Find code examples and implementation patterns
- Research best practices and architectural approaches
- Look up error messages and debugging solutions

## Best Practices
- Verify information from multiple sources when possible
- Prefer official documentation over third-party sources
- Check publication dates to ensure information is current
- Summarize findings clearly and cite sources
