# Sandbox Coding

Proficiency in writing, executing, and debugging code within isolated sandbox environments.

## Capabilities
- Write code in multiple languages (Python, JavaScript, Go, Rust, etc.)
- Execute code safely in containerized sandbox environments
- Read and write files within the sandbox filesystem
- Run shell commands for building, testing, and debugging
- Iteratively refine code based on execution results

## Best Practices
- Always test code in the sandbox before delivering final results
- Handle errors gracefully and provide clear error messages
- Write clean, well-structured code with appropriate comments
- Use version control best practices when managing project files
