# 代码工程师 (Engineer) 招聘需求草案

## 1. 岗位当前JD草案
- **岗位名称**：初级工程师 (Junior Engineer)
- **所属部门**：技术研发部 (Engineering)
- **岗位职责**：
  1. 参与公司核心业务系统（包括AI员工系统）的后端与部分前端开发工作。
  2. 配合团队维护现有技术基础设施，确保 OpenSandbox 及相关服务的稳定运行。
  3. 独立完成模块设计、编码、测试和文档编写。
  4. 践行公司“狼性文化”，快速响应业务需求，保障交付质量。
- **任职要求**：
  1. 熟练掌握 Python（核心要求），熟悉 JavaScript 和 Shell 脚本。
  2. 熟悉主流 Web 框架和数据库系统。
  3. 具备良好的工程素养和代码规范，有独立解决复杂技术问题的能力。
  4. 认同“实事求是，求是创新”的价值观，具备极强独立作战能力的复合型人才优先。

## 2. 团队当前技术栈/基础设施
- **编程语言**：Python（主力后端/AI逻辑）、JavaScript（前端/部分脚本）、Shell（自动化运维）
- **基础设施/云服务**：基于 OpenSandbox 的容器化安全执行环境
- **数据库/中间件**：通用关系型数据库（如 PostgreSQL/MySQL）及常见缓存/消息队列
- **CI/CD**：基于现有自动化工作流的代码测试与部署机制

## 3. 未来3-6个月项目/交付需求
- **系统核心能力升级**：扩展 AI 系统支持的工具链，提升多智能体协作效率。
- **业务流程自动化扩充**：支撑并落地更多复杂的业务自动化场景（覆盖HR、运营、销售等）。
- **稳定性与安全性加固**：优化现有代码架构，增强沙箱执行环境的安全性与异常处理机制。

## 4. 预期级别与薪酬预算想法
- **预期级别**：Lv.1 初级（根据公司职级体系，所有新入职员工一律从 Lv.1 开始，后续通过连续3季度3.75绩效晋升）。
- **花名要求**：入司后分配两个中文字的武侠风花名（如：青锋、破军）。
- **薪酬预算**：按市场初级/中级开发工程师标准（建议区间：15k - 25k RMB/月，具体依据候选人实际能力与公司财务状况由 COO/CEO 最终定夺）。
