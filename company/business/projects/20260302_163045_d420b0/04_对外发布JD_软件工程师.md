# 软件工程师（后端/全栈，后端偏重）

**地点：**远程/城市（根据实际情况安排）
**工作方式：**远程/混合/线下
**薪资范围：**国内全职 25k–40k RMB/月（13–15薪折算），视能力和经验而定，提供具有市场竞争力的薪资及期权激励；海外/远程 $45k–$85k/年 或 $25–$55/h

### 我们在做什么
One Man Company 是小团队高密度交付型公司。你将直接与 CEO/核心负责人协作，端到端交付业务功能，并持续提升工程质量。

### 你要负责
- 参与需求澄清与技术方案设计，把不确定问题拆解成可执行任务。
- 开发并维护后端服务与 API（含鉴权、分页、幂等、错误码规范）。
- 数据建模与数据库开发：表结构设计、迁移脚本、查询优化。
- 工程质量：单元/集成测试、代码规范、CI 通过、文档沉淀。
- 上线与运维准备：Docker 化、发布/回滚、日志与基础监控，能独立排障。

### 你入职后 3–6 个月的交付目标
- **第1个月内**：完成开发环境跑通、能独立交付 1 个小功能（含测试/部署/文档）。
- **第2–3个月**：独立负责 1 个核心模块/服务（API+DB），完成上线与稳定性保障，做到可观测、可回滚。
- **第4–6个月**：推进工程化（测试覆盖、CI、发布流程）并完成 1 次性能/稳定性改进。

### 我们希望你具备（Must-have）
- 熟练 **Python 或 TypeScript/Node.js**（至少一种能熟练生产落地）。
- 熟悉 REST API 设计与常见工程实践（鉴权、幂等、错误处理、日志）。
- 熟练 SQL（PostgreSQL/MySQL 任一）：能做表设计、事务理解、基础优化。
- 熟练 Git 协作（PR、review、rebase），有测试意识与实践（pytest/jest）。
- 强 ownership：能用文档/PR 说明方案与进展，能明确风险并主动升级沟通。

### 当前/优先技术栈（用于匹配）
- Python/FastAPI（主）或 Node(Express/NestJS)
- PostgreSQL（优先）/ MySQL
- Docker, GitHub Actions (CI), Linux

### 加分项（Nice-to-have）
- FastAPI/Django/NestJS 实战经验。
- Redis/消息队列/限流熔断等稳定性建设经验。
- CI/CD、云服务（AWS/GCP/阿里云）与可观测性实践。
- LLM 应用（RAG/向量库/评测）经验。

### 应聘材料
- 简历 + GitHub/作品链接（若有） + 1 个最能代表你交付能力的项目说明（2–3 段）。
