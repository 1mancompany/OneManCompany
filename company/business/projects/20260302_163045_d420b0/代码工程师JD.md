# 代码工程师 JD

## 职责
1. 负责公司核心业务系统的后端开发、架构设计与性能优化。
2. 参与需求分析、技术选型，独立完成模块级的设计与开发工作。
3. 保证代码质量，编写单元测试，参与代码审查（Code Review）。
4. 探索并应用 AI/LLM 技术，赋能业务场景，提升产品竞争力。
5. 解决线上复杂问题，保障系统高可用性和稳定性。

## 必备技能
1. 熟练掌握 Python 或 JavaScript，具备扎实的编程基础和良好的代码规范。
2. 熟悉常用框架（如 FastAPI 等），了解其底层原理。
3. 熟悉数据库设计与优化（如 MySQL, PostgreSQL, Redis 等）。
4. 熟练掌握常用数据结构与算法，具备良好的逻辑思维和解决问题的能力。
5. 具备良好的工程化能力，熟悉 Git、CI/CD 等开发流程。

## 加分技能
1. 有 AI/LLM 相关开发经验（如 LangChain, LlamaIndex, Prompt Engineering 等）。
2. 具备前端开发经验，了解 React/Vue 等主流框架。
3. 有分布式系统、高并发场景处理经验。
4. 熟悉容器化技术（Docker, Kubernetes）。

## 工作形式/地点
- 远程/线下：结合公司实际情况安排。
- 创业公司节奏，要求具备极强独立作战能力和自我驱动力。

## 薪资范围
- 面议（视能力和经验而定，提供具有市场竞争力的薪资及期权激励）。

## 面试流程建议
1. 简历筛选：关注技术栈匹配度、项目经验及工程化能力。
2. 一面（技术初试）：基础知识（语言特性、数据结构与算法）、项目经历深挖、编码能力测试。
3. 二面（技术复试）：架构设计、复杂问题解决思路、AI/LLM 相关经验探讨。
4. 三面（HR/CEO 面）：文化契合度（创业精神、独立作战能力）、职业规划、薪酬沟通。
