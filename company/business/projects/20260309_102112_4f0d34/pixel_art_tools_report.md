# Pixel Art 生成工具调研报告

> 调研日期：2026-03-09

## 一、AI 驱动的 Pixel Art 生成工具

### 1. PixelLab ⭐ 推荐
- **价格**：$12-50/月
- **核心功能**：文本/图片生成像素画，骨骼动画，8方向角色生成，等距视角支持，Aseprite 插件
- **API**：✅ 完整 REST API + MCP 支持（可集成 AI 编程助手）
- **优点**：功能最全面，游戏资产专用设计，API 成熟，支持自动化流水线
- **缺点**：月费较高，风格偏向 RPG/冒险类
- **适用场景**：等距/俯视角 RPG 游戏资产批量生产

### 2. Retro Diffusion ⭐ 推荐
- **价格**：约 $0.06/张（按量计费）
- **核心功能**：4个专用模型（Plus/Fast/Tile/Animation），16x16 到 256x256px 输出
- **API**：✅ 通过自有平台或 Runware 提供 API
- **优点**：性价比最高，适合批量生成，贴图/纹理专用模型
- **缺点**：需要更多手动后期调整
- **适用场景**：大批量 tile/纹理生成

### 3. Scenario
- **价格**：$15/月（Pro）
- **核心功能**：用 10-50 张参考图训练自定义模型，保持风格一致性
- **API**：✅ API-first 设计，Unity/Unreal 插件
- **优点**：风格控制最强，内含 Retro Diffusion 模型
- **缺点**：需要准备训练数据集
- **适用场景**：需要严格风格统一的项目

### 4. SEELE
- **价格**：有免费层
- **核心功能**：快速精灵图生成（约10秒/张）
- **API**：⚠️ 有限
- **优点**：速度快，免费可用
- **缺点**：API 能力有限，不适合深度集成

### 5. Pixelicious
- **价格**：免费
- **核心功能**：图片转像素画（仅转换，不生成原创内容）
- **API**：✅ 但仅限转换功能
- **优点**：免费，转换效果好
- **缺点**：不能生成原创资产

## 二、传统 Pixel Art 工具

| 工具 | 价格 | 特点 | 自动化能力 |
|------|------|------|------------|
| **Aseprite** | $20（一次性） | 业界标准，Lua 脚本扩展，丰富插件生态 | ✅ Lua 脚本 |
| **LibreSprite** | 免费 | Aseprite 早期分支 | ❌ 落后 |
| **Piskel** | 免费 | 浏览器端，入门友好 | ❌ 无 |

## 三、推荐方案：AI 游戏开发流水线

```
PixelLab/Retro Diffusion API  →  Aseprite (Lua脚本精修)  →  游戏引擎导入
      (AI生成)                    (自动化后处理)              (直接使用)
```

**最佳组合推荐**：
1. **PixelLab** 作为主力生成工具（API 完善 + MCP 支持，与我们的 AI 工作流天然契合）
2. **Retro Diffusion** 作为补充（批量 tile/纹理生成，性价比高）
3. **Aseprite** 作为精修和导出环节（Lua 脚本自动化 + 两者都有 Aseprite 插件）

## 四、关键结论

- **PixelLab 的 MCP 支持**是最大亮点，可以让 AI 编程助手直接调用像素画生成能力，非常适合我们的 AI 驱动开发模式
- 按量计费的 Retro Diffusion 适合探索阶段，PixelLab 月费适合正式生产阶段
- Aseprite 是必备的精修工具，$20 一次性购买非常划算
