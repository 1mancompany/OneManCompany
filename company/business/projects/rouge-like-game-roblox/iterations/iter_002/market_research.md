# Roblox Rogue-like游戏市场调研报告

## 一、Roblox平台热门Rogue-like/Roguelite游戏分析

### 1. Deepwoken（深渊觉醒）

| 指标 | 数据 |
|------|------|
| 开发商 | Monad Studios |
| 访问量 | 10亿+ |
| 付费模式 | 付费准入（400 Robux），Roblox最成功的付费游戏 |
| 类型 | 开放世界 + Roguelike + Soulslike ARPG |

**核心玩法：**
- 战斗系统围绕姿态值（Posture）和血量的平衡，通过弹反（Parry）机制打空敌人姿态
- 每升15级可从随机"卡牌手牌"中选择天赋（Talents）和咒术（Mantras）
- 永久死亡（Permadeath）机制，死亡后角色完全重置
- PvE为主，鼓励PvP遭遇战

**留存机制：**
- 高难度+永久死亡制造"再来一局"的成瘾循环
- 深度Build系统（天赋+咒术组合）鼓励多次尝试不同流派
- 开放世界探索提供持续发现感

**变现模式：**
- 400 Robux付费准入（一次性买断）
- 无侵入性内购，靠品质驱动口碑

**启示：** 证明了Roblox玩家愿意为高品质硬核体验付费。Soulslike+Roguelike组合在平台上有强劲需求。

---

### 2. Rogue Lineage（盗贼血脉）

| 指标 | 数据 |
|------|------|
| 开发商 | Monad Studios |
| 上线时间 | 2019年3月 |
| 付费模式 | 付费准入（200 Robux） |
| 类型 | Roguelike ARPG + PvP |

**核心玩法：**
- 每个角色限3条命，死亡后传承部分属性给"继承者"
- 完全死亡（Wipe）后失去一切：经验、技能、名字、武器、背包
- 职业进阶系统：Base → Super → Ultra → Uber，需要在不同世界完成
- 随机生成角色种族和名字（付费可自选）

**留存机制：**
- 继承系统让每次死亡都有意义（部分法力传承）
- 高风险PvP制造紧张感和社交冲突
- 多职业路线鼓励反复游玩

**变现模式：**
- 200 Robux付费准入
- 微交易：改名/改种族
- 额外生命（Robux购买，限一次）

**启示：** 继承（Legacy）系统是Roguelike在Roblox上的创新突破——让永久死亡不那么令人沮丧，同时保持紧张感。

### 3. DOORS（门）

| 指标 | 数据 |
|------|------|
| 开发商 | LSPLASH |
| 访问量 | 50亿+（截至2025约15亿，持续增长） |
| 付费模式 | 免费游玩 + GamePass |
| 类型 | 第一人称恐怖 + Roguelite |

**核心玩法：**
- 第一人称视角穿越100扇程序化生成的门/房间
- 每个房间隐藏不同的"实体"（怪物），需要学习应对策略
- 1-6人合作（最多50人但不推荐）
- 多层楼（Hotel、Mines）+ 隐藏区域（The Backdoor、The Rooms）

**留存机制：**
- 房间随机排列保证每局不同
- 隐藏区域和秘密激发探索欲
- 社交恐怖体验（一起被吓到）提升分享传播
- 定期更新新楼层/实体

**变现模式：**
- 免费准入吸引海量玩家
- GamePass：复活币、指南针等辅助道具
- 预计收入超300万美元

**启示：** 恐怖+Roguelite是一个被验证的爆款公式。程序化房间生成+实体学习曲线=极高重玩价值。社交恐怖体验是天然的传播引擎。

### 4. Dungeon Quest（地牢任务）

| 指标 | 数据 |
|------|------|
| 开发商 | Voldex |
| 访问量 | 23亿+ |
| 上线时间 | 2018年9月 |
| 类型 | 地牢爬行 ARPG |

**核心玩法：**
- 17个独特地牢，每个有不同敌人和Boss
- 职业系统：战士、法师、猎人等
- 稀有装备掉落系统（刷装备循环）
- 多人组队合作通关

**留存机制：**
- 装备稀有度驱动的"刷刷刷"循环（白→绿→蓝→紫→金→传说）
- 难度递增的地牢进度线
- 排行榜和竞速

**变现模式：**
- 免费游玩
- GamePass：双倍经验、双倍金币、VIP专属区域
- 外观道具

**启示：** 经典地牢爬行+装备驱动在Roblox上经久不衰。简单易懂的核心循环（打怪→掉装备→变强→打更难的怪）是留存基石。

### 5. Vesteria

| 指标 | 数据 |
|------|------|
| 类型 | MMORPG + 地牢 |
| 特点 | 三职业系统（战士/法师/猎人） |
| 付费模式 | 曾付费准入，后转免费 |

**核心玩法：**
- 经典MMORPG框架：任务系统、地牢副本、公会
- 三大武器职业允许不同玩法风格和Build定制
- 探索导向的开放世界

**留存机制：**
- 职业深度和Build多样性
- 社交系统（公会、交易）
- 持续内容更新

**变现模式：**
- 转为免费后通过GamePass和外观变现

**启示：** 职业系统深度是Roblox RPG长寿命的关键。社交+公会系统极大提升留存。

### 6. Pilgrammed（朝圣者）

| 指标 | 数据 |
|------|------|
| 类型 | 开放世界深度RPG |
| 特点 | Roblox上最大地图之一，深度锻造系统 |

**核心玩法：**
- 超大开放世界探索，大量隐藏秘密
- 深度武器/护甲锻造系统
- Boss战（每个Boss有独特技能和策略）
- 击败Boss获得独占武器/护甲/资源
- 硬核模式（Hardcore Mode）可选

**留存机制：**
- 海量探索内容和秘密
- 锻造系统的深度定制
- 公会系统（Guilds）
- 频繁更新（裂隙、新区域）

**启示：** 深度制作系统+探索驱动在Roblox上有强劲需求。证明了"非速食"体验在平台上也能成功。

## 二、Roblox平台2025-2026热门趋势分析

### 2.1 平台整体数据

- **日活访问量排名（按类型）：** 角色扮演/模拟（116M日访问）> 模拟（110M）> 平台跳跃（70M）> 生存（70M）> 动作（59M）
- **历史最高同时在线：** 2025年9月 Steal a Brainrot 超过2500万同时在线
- **开发者生态：** 累计为开发者创造超27亿美元收入

### 2.2 六大关键趋势

#### 趋势1：深度系统 > 简单循环
成功游戏正从简单的点击/AFK循环转向深度系统设计。Fisch（钓鱼+探索+收集）、Grow a Garden（种植+经济+交易）证明复杂机制在Roblox上不仅可行，还能带来更高收入（Grow a Garden月收入1200万美元）。

#### 趋势2：社交优先的游戏设计
Roblox已从"游戏平台"转型为"社交优先的游戏生态"。最成功的体验感觉像"活的在线世界"而非传统游戏。社交Hangout空间传播趋势的速度超过任何玩法驱动的类型。

#### 趋势3：移动F2P变现模型的全面渗透
计时器、加速、扭蛋/抽卡、外观、通行证——移动端F2P验证过的变现策略已成为Roblox顶级游戏的标配。

#### 趋势4：频繁更新 = 持续留存
周更新、月扩展、季节性活动是维持玩家基数的关键。Fisch的"每周更新+每月扩展"模式是成功范例。

#### 趋势5：Anime/IP融合
动漫元素在Roblox上有极强号召力。Blox Fruits（海贼王灵感，529亿访问量）、Anime Last Stand、Anime Defenders都证明了动漫风格和玩法融合的市场潜力。

#### 趋势6：激励视频广告
2025年推出的Rewarded Video Ads为开发者提供了新收入来源——玩家选择观看短广告换取游戏内奖励。

## 四、市场空白与机会点

### 4.1 已验证的成功要素
1. **永久死亡+继承系统**（Deepwoken/Rogue Lineage验证）
2. **程序化生成内容**（DOORS验证）
3. **深度Build系统**（Deepwoken验证）
4. **社交合作体验**（DOORS/Dungeon Quest验证）
5. **装备驱动循环**（Dungeon Quest验证）

### 4.2 市场空白
1. **缺少"中等硬核"Roguelike** — Deepwoken/Rogue Lineage太硬核，DOORS太休闲，中间地带空缺
2. **缺少PvE合作Roguelike** — 现有头部要么重PvP，要么是单人/轻合作
3. **缺少Roguelike+动漫风格融合** — 动漫是Roblox最强IP标签，但没有头部Roguelike采用
4. **缺少"一局15-20分钟"的快节奏Roguelike** — 类似Hades/Vampire Survivors的快节奏Run在Roblox上缺位
5. **缺少Roguelike+社交展示** — 没有头部作品将Roguelike战利品与社交展示（大厅、排行榜、外观）深度结合

### 4.3 创新融合方向建议

**推荐方向：快节奏PvE合作Roguelike + 动漫风格 + 社交大厅**

理由：
- 填补"中等硬核"+"快节奏Run"的市场空白
- 利用动漫风格的Roblox生态号召力
- PvE合作降低门槛、提升社交传播
- 社交大厅承接Roblox"社交优先"趋势
- 15-20分钟单局适合Roblox用户碎片化游玩习惯

---

## 五、关键结论

1. **Roguelike在Roblox上已被验证**，但头部竞品集中在"超硬核"赛道
2. **中等难度+快节奏+合作** 是最大的未填充市场空白
3. **动漫美术风格** 是Roblox生态的流量密码
4. **免费准入+GamePass/DevProducts** 是主流变现路径，付费准入仅适合已有口碑的工作室
5. **频繁更新+社交功能** 是长期留存的核心驱动力
6. **程序化生成+Build深度** 是Roguelike重玩价值的技术基石
