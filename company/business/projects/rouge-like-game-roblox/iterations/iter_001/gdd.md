# Game Design Document: 《Soul Rift》

> Roblox Roguelike ARPG | Version 1.0 | 2026-03-10

---

## 1. 游戏概述

### 1.1 一句话描述
**Soul Rift** 是一款动漫风格的快节奏PvE合作Roguelike ARPG，1-4人组队深入程序化生成的"灵魂裂隙"，在15-20分钟的单局中选择Build、击败Boss、收集战利品，然后在社交大厅展示你的成就。

### 1.2 核心定位
- **类型：** Roguelike ARPG（类Hades快节奏 + 类Dungeon Quest装备驱动）
- **目标平台：** Roblox（PC + Mobile优先）
- **目标受众：** 10-18岁Roblox核心用户，喜欢动漫和合作PvE
- **单局时长：** 15-20分钟
- **队伍规模：** 1-4人
- **付费模式：** 免费游玩 + GamePass + DevProducts

### 1.3 核心卖点（USP）
1. **快节奏动作Roguelike** — 区别于Deepwoken的慢节奏Soulslike，更接近Hades的爽快战斗
2. **动漫美术风格** — 利用Roblox生态最强流量标签
3. **PvE合作优先** — 降低门槛，提升社交传播
4. **社交大厅系统** — 战利品展示、排行榜、角色外观秀场
5. **中等难度曲线** — 填补"超硬核"与"纯休闲"之间的市场空白

### 1.4 设计参考
| 参考游戏 | 借鉴要素 |
|---------|---------|
| Hades | 快节奏战斗节奏、Boon选择系统、叙事与Roguelike融合 |
| Vampire Survivors | 简单操作+复杂Build涌现 |
| DOORS (Roblox) | 程序化房间生成、合作体验 |
| Dungeon Quest (Roblox) | 装备驱动循环、GamePass变现 |
| Deepwoken (Roblox) | 天赋卡牌选择系统 |

## 2. 核心玩法循环

### 2.1 单局循环（Run Loop）

```
准备阶段（大厅选角色/装备）
    ↓
进入裂隙（程序化生成地牢）
    ↓
┌─→ 进入房间 → 清怪战斗 → 获得奖励
│       ↓
│   选择Buff（从3选1天赋卡中选择）
│       ↓
│   选择路径（分支选择下一个房间类型）
│       ↓
└── 重复 8-12 个房间
    ↓
Boss战（当前层Boss）
    ↓
通关/失败 → 结算（经验、装备、货币）
    ↓
返回社交大厅
```

### 2.2 Meta循环（跨局进度）

```
单局获得 → 经验值 + 装备 + 灵魂碎片（货币）
    ↓
角色永久成长：解锁新职业/技能树/被动
    ↓
装备系统：收集、强化、合成
    ↓
赛季通行证：限时外观/表情/特效
    ↓
社交系统：排行榜、成就展示、交易
```

### 2.3 房间类型

| 房间类型 | 图标 | 说明 |
|---------|------|------|
| 战斗房 | ⚔️ | 清除所有敌人获得奖励 |
| 精英房 | 💀 | 精英怪，更难但奖励更好 |
| 宝箱房 | 🎁 | 无战斗，开宝箱获得装备/货币 |
| 商店房 | 🛒 | 用局内货币购买天赋/回复/装备 |
| 休息房 | 💚 | 恢复生命，可选择强化现有天赋 |
| Boss房 | 👑 | 每层终点，击败后进入下一层或通关 |
| 挑战房 | ⚡ | 限时/限制条件战斗，高风险高回报 |
| 秘密房 | ❓ | 低概率出现，独特奖励和彩蛋 |

## 3. 角色/职业系统

### 3.1 职业设计

初始解锁2个职业，其余通过游玩解锁：

#### 🗡️ 剑士（Blade Dancer）
- **定位：** 近战 / 中等难度
- **特点：** 快速连击、闪避反击、剑气波
- **被动：** 连击数越高，攻击速度和伤害越高
- **解锁条件：** 初始可用

#### 🔮 咒术师（Soul Caster）
- **定位：** 远程 / 入门友好
- **特点：** 灵魂弹、范围法术、护盾
- **被动：** 施法后短时间内获得灵魂护盾
- **解锁条件：** 初始可用

#### 🥊 拳师（Iron Fist）
- **定位：** 近战 / 高难度高回报
- **特点：** 近身格斗、抓投、蓄力重击
- **被动：** 完美闪避后下次攻击暴击
- **解锁条件：** 通关1次任意难度

#### 🏹 猎手（Shadow Hunter）
- **定位：** 远程 / 高机动
- **特点：** 弓箭、陷阱、隐身暗杀
- **被动：** 从隐身状态攻击造成双倍伤害
- **解锁条件：** 通关5次

#### 🛡️ 守护者（Aegis）
- **定位：** 坦克 / 团队核心（MVP解锁后）
- **特点：** 盾牌、嘲讽、队友护盾
- **被动：** 受到伤害为队友生成护盾碎片
- **解锁条件：** 合作通关10次

#### ⚡ 雷灵（Storm Spirit）
- **定位：** 高速 / 范围控制（MVP解锁后）
- **特点：** 闪电链、瞬移、范围电场
- **被动：** 移动积攒电荷，释放技能消耗电荷增幅效果
- **解锁条件：** 累计击败1000敌人

### 3.2 技能系统

每个职业有：
- **3个主动技能**（可通过天赋卡强化/改变效果）
- **1个终极技能**（蓄满能量条后释放）
- **1个被动天赋**（通过跨局升级强化）

技能操作设计考虑Mobile端：
- 左侧虚拟摇杆移动
- 右侧3个技能按钮 + 1个终极按钮
- 自动普攻（可关闭）
- 闪避按钮

## 4. 天赋卡系统（局内Build构建）

### 4.1 核心机制

每清完一个战斗房间，从3张随机天赋卡中选择1张。天赋卡决定本局的Build方向。

### 4.2 天赋卡类型

| 稀有度 | 颜色 | 出现概率 | 效果强度 |
|--------|------|---------|---------|
| 普通 | 白色 | 50% | 基础强化 |
| 稀有 | 蓝色 | 30% | 显著强化 |
| 史诗 | 紫色 | 15% | 改变技能效果 |
| 传说 | 金色 | 5% | 定义Build方向 |

### 4.3 天赋卡示例

**通用天赋：**
- 【生命汲取】攻击回复1%最大生命值（普通）
- 【连锁闪电】击杀敌人时对周围敌人造成50%伤害（稀有）
- 【时间扭曲】闪避时减慢周围敌人50%持续2秒（史诗）
- 【不死之魂】死亡时原地复活，60秒内攻击力+100%（传说）

**剑士专属：**
- 【风之剑】剑气波范围+50%（普通）
- 【影分身斩】连击结束时召唤分身攻击（稀有）
- 【万剑归宗】终极技能改为召唤剑阵持续攻击区域（史诗）
- 【剑意觉醒】连击数永不中断，每10连击+5%攻击力（传说）

**天赋卡联动（Synergy）：**
同时持有特定天赋卡组合触发额外效果：
- 【生命汲取】+【连锁闪电】→ 连锁闪电也能回血
- 【时间扭曲】+【影分身斩】→ 分身在时间减速区域内攻击速度不受影响

## 5. 敌人与Boss设计

### 5.1 普通敌人（按区域主题）

#### 第一层：幽灵废墟
| 敌人 | 行为模式 | 难度 |
|------|---------|------|
| 游荡幽灵 | 缓慢追踪，近身爆炸 | ★ |
| 骨弓手 | 远程射击，低血量 | ★ |
| 暗影骑士 | 冲锋+旋转斩 | ★★ |
| 符文守卫 | 护盾+反弹弹射物 | ★★★ |

#### 第二层：熔岩神殿
| 敌人 | 行为模式 | 难度 |
|------|---------|------|
| 火焰精灵 | 远程火球+地面火焰 | ★★ |
| 岩石巨人 | 慢速高伤，地面震击 | ★★ |
| 熔岩蛇 | 快速穿地攻击 | ★★★ |
| 炎魔祭司 | 召唤小兵+范围火雨 | ★★★★ |

#### 第三层：虚空深渊
| 敌人 | 行为模式 | 难度 |
|------|---------|------|
| 虚空行者 | 瞬移+背刺 | ★★★ |
| 裂隙触手 | 地面陷阱+抓取 | ★★★ |
| 混沌编织者 | 改变房间地形+召唤 | ★★★★ |
| 虚空吞噬者 | 吸引+范围压制 | ★★★★★ |

### 5.2 Boss设计

每层1个Boss，有多阶段机制：

#### Boss 1：亡灵领主 · 拉扎尔
- **阶段1（100%-60%HP）：** 剑斩连击+召唤骨兵
- **阶段2（60%-30%HP）：** 脱离肉体变为灵魂形态，全屏追踪攻击
- **阶段3（30%-0%HP）：** 双形态切换，剑与灵魂同时攻击
- **掉落：** 亡灵系列装备、灵魂碎片

#### Boss 2：炎狱之王 · 伊格尼斯
- **阶段1：** 巨型火锤砸地+岩浆喷射
- **阶段2：** 竞技场地面逐渐被岩浆覆盖，安全区缩小
- **阶段3：** 变身巨型火凤凰，飞行+俯冲攻击
- **掉落：** 炎狱系列装备、稀有天赋卡碎片

#### Boss 3：虚空主宰 · 奈克罗斯
- **阶段1：** 空间扭曲——随机传送玩家位置，分割团队
- **阶段2：** 复制玩家的影子与玩家对战
- **阶段3：** 虚空崩塌，限时击杀否则全灭
- **掉落：** 虚空系列装备、传说天赋卡碎片

#### 隐藏Boss：深渊造物主 · ???
- **解锁条件：** 在单局内收集3个隐藏碎片（分布在秘密房间）
- **特点：** 随机使用前三个Boss的技能+独特机制
- **掉落：** 专属传说装备、限定外观

## 6. 物品/装备/技能系统

### 6.1 装备系统

#### 装备槽位
| 槽位 | 说明 |
|------|------|
| 武器 | 决定攻击方式和基础伤害 |
| 头盔 | 提供生命值和特殊效果 |
| 护甲 | 提供防御力和职业特效 |
| 饰品 | 提供独特被动效果（2个槽位） |

#### 装备稀有度
| 稀有度 | 颜色 | 来源 | 词条数 |
|--------|------|------|--------|
| 普通 | 白色 | 战斗掉落 | 0 |
| 优质 | 绿色 | 战斗掉落 | 1 |
| 稀有 | 蓝色 | 精英/宝箱 | 2 |
| 史诗 | 紫色 | Boss掉落 | 3 |
| 传说 | 金色 | Boss稀有掉落 | 4 |
| 神话 | 红色 | 隐藏Boss/合成 | 5 + 独特效果 |

#### 装备词条（随机生成）
- 攻击力 +X%
- 生命值 +X
- 暴击率 +X%
- 冷却缩减 +X%
- 灵魂汲取（攻击回血）
- 雷电附魔（攻击附带雷电伤害）
- 移速 +X%

### 6.2 装备强化系统

使用**灵魂碎片**（通用货币）在大厅的铁匠处强化装备：
- +1到+5：100%成功率，逐级增加费用
- +6到+10：成功率递减（80%→40%），失败不掉级但消耗材料
- +10以上：需要特殊材料（Boss掉落）

### 6.3 装备合成

3件相同稀有度的装备 → 1件更高稀有度的装备（随机词条）

## 7. 地图/关卡生成规则

### 7.1 裂隙结构

每局裂隙包含3层，每层8-12个房间 + 1个Boss房：

```
层1（幽灵废墟）: 8-10房间 → Boss
    ↓
层2（熔岩神殿）: 9-11房间 → Boss
    ↓
层3（虚空深渊）: 10-12房间 → Boss
    ↓
（可选）隐藏层: 5房间 → 隐藏Boss
```

### 7.2 房间生成算法

每层开始时生成房间序列：
1. **保证规则：** 每层至少1个商店房、1个休息房、1个精英房
2. **首房规则：** 第一个房间一定是战斗房（简单敌人，热身）
3. **Boss前规则：** Boss房前一定是休息房（准备时间）
4. **分支选择：** 每2-3个房间后出现路径分支（2选1），玩家选择下一个房间类型
5. **秘密房规则：** 每层10%概率出现秘密房入口（隐藏在战斗房/宝箱房中）
6. **难度曲线：** 房间内敌人数量和强度随房间序号线性增加

### 7.3 房间布局模板

每个房间从预制模板库中随机选择布局：
- **开阔型：** 大空间，适合远程职业
- **走廊型：** 狭窄通道，近战优势
- **多平台型：** 高低差平台，机动性重要
- **竞技场型：** 圆形封闭空间，Boss战专用
- **迷宫型：** 多路径，可能有隐藏宝箱

### 7.4 难度系统

| 难度 | 解锁条件 | 敌人倍率 | 奖励倍率 | 特殊规则 |
|------|---------|---------|---------|---------|
| 普通 | 默认 | x1 | x1 | 无 |
| 困难 | 通关普通 | x1.5 | x1.5 | 精英房数量+1 |
| 噩梦 | 通关困难 | x2.5 | x2.5 | 无休息房 |
| 深渊 | 通关噩梦 | x4 | x4 | 永久死亡（单局内无复活） |
| 无尽 | 通关深渊 | 递增 | 递增 | 无限层数，排行榜竞速 |

### 8.2 UX设计原则

1. **Mobile优先：** 所有交互适配触屏，按钮大小≥44px
2. **信息分层：** 战斗中只显示关键信息，详细数据在暂停/大厅查看
3. **视觉反馈：** 伤害数字飘字、暴击特效、技能命中震屏
4. **新手引导：** 首次游玩有引导教程裂隙（3个房间+简化Boss）
5. **无障碍：** 支持色盲模式、伤害数字大小可调

## 9. 变现设计

### 9.1 GamePass（一次性购买）

| GamePass | 价格(Robux) | 内容 |
|----------|------------|------|
| VIP通行证 | 399 | 每日额外奖励、VIP大厅区域、专属表情 |
| 双倍经验 | 199 | 永久双倍角色经验 |
| 双倍灵魂碎片 | 199 | 永久双倍货币掉落 |
| 装备收藏家 | 149 | 额外50格装备仓库 |
| 天赋大师 | 249 | 天赋卡选择从3选1变为4选1 |
| 自动拾取 | 99 | 自动拾取掉落物 |

### 9.2 DevProducts（可重复购买）

| 商品 | 价格(Robux) | 内容 |
|------|------------|------|
| 灵魂碎片包(小) | 49 | 500灵魂碎片 |
| 灵魂碎片包(大) | 199 | 2500灵魂碎片 |
| 复活币 x3 | 29 | 局内死亡可消耗复活 |
| 天赋刷新 x5 | 49 | 重新随机天赋卡选项 |
| 外观宝箱 | 79 | 随机获得1个外观道具 |

### 9.3 赛季通行证（Season Pass）

- **免费轨道：** 基础奖励（货币、普通外观）
- **付费轨道（299 Robux）：** 限定外观、表情、特效、装备外观
- **每赛季8周**，通过每日/每周任务推进进度
- 赛季结束后限定物品不再获取（FOMO驱动）

### 9.4 Rewarded Video Ads

- 每局结束可选择观看广告获得额外奖励（+50%结算奖励）
- 大厅中每30分钟可观看广告获得小额灵魂碎片
- 不影响核心体验，纯粹可选

### 9.5 变现原则

1. **绝不卖数值（Pay-to-Win）**：所有付费内容为便利性或外观
2. **免费玩家体验完整**：所有内容可通过游玩获得
3. **付费加速但不跳过**：双倍经验加速成长，但不跳过游玩过程

## 10. 社交系统

### 10.1 社交大厅
- 最多50人共享大厅实例
- 可查看其他玩家角色、装备、成就
- 大厅内有NPC（铁匠、商人、任务板）
- 聊天系统（Roblox内置）

### 10.2 排行榜系统
- **本赛季排行：** 最高层数/难度通关
- **速通排行：** 最快通关时间（按难度分榜）

## 13. 技术架构概要

### 13.1 Roblox技术栈

| 层级 | 位置 | 职责 |
|------|------|------|
| Server | ServerScriptService | 战斗计算、掉落生成、进度存储 |
| Shared | ReplicatedStorage | 数据表、技能配置、UI模板 |
| Client | StarterPlayerScripts | 输入处理、特效渲染、UI逻辑 |

### 13.2 关键技术要点
- **服务端权威：** 所有伤害、掉落、进度计算在服务端执行，防作弊
- **程序化生成：** 服务端生成房间序列，客户端根据种子加载对应模板
- **数据存储：** DataStore2 或 ProfileService 存储玩家永久进度
- **网络优化：** 减少RemoteEvent频率，使用CFrame插值平滑移动

## 14. 风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|---------|
| 战斗手感不佳 | 核心体验崩塌 | MVP阶段反复测试打击感，参考Hades战斗节奏 |
| 程序化生成单调 | 重玩价值降低 | 增加房间模板多样性，引入随机事件 |
| 变现过于激进 | 玩家流失 | 严守"不卖数值"原则，参考DOORS的轻变现 |
| 多人同步卡顿 | 合作体验差 | 服务端权威+客户端预测，限4人上限 |
| 动漫风格版权 | 法律风险 | 创造原创角色/世界观，不直接引用现有IP |
| 内容消耗过快 | 玩家流失 | 每2周小更新，每月大更新，赛季驱动长线 |
