# 面试题库与评估表

## 面试题库
### 1. 基础编程与算法
- 请反转一个单链表。
- 解释 Python 中的装饰器，并手写一个计算函数执行时间的装饰器。
- 什么是闭包？请举例说明。

### 2. 框架与工程化
- 描述 RESTful API 的设计原则。
- 数据库索引的原理是什么？什么情况下索引会失效？
- Git 中 merge 和 rebase 的区别是什么？

### 3. AI/大模型相关（加分项）
- 你如何理解 Prompt Engineering？在实际中有过哪些应用？
- 调用大模型 API 时遇到超时或速率限制（Rate Limit）通常如何处理？

## 面试评估表
| 评估维度 | 评分标准 (1-5分) | 权重 | 得分 | 备注 |
|---|---|---|---|---|
| 代码基础 | 1:差 3:合格 5:极佳 | 30% | | 考察算法、数据结构、语言特性 |
| 工程能力 | 1:差 3:合格 5:极佳 | 30% | | 考察架构设计、数据库、版本控制 |
| 问题解决 | 1:差 3:合格 5:极佳 | 20% | | 考察面对未知问题的拆解与分析能力 |
| 沟通与协作 | 1:差 3:合格 5:极佳 | 10% | | 考察表达能力、团队合作精神 |
| 文化契合度 | 1:差 3:合格 5:极佳 | 10% | | 考察是否符合“狼性文化”、实事求是 |

**录用门槛**:
- 总分 >= 3.5 分（满分5分）。
- 任何单项不得低于 3 分。
- 核心代码基础与工程能力必须达到合格及以上。
