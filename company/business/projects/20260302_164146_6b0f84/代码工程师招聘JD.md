# 代码工程师（初级）招聘要求

## 职责
- 负责公司核心业务系统的后端/前端代码编写与维护。
- 参与自动化工作流、AI Agent 相关的工程化落地。
- 编写高质量、可维护的代码，并完善相关文档。
- 解决开发过程中的技术难题，保证系统稳定运行。

## 必备技能
- 熟练掌握 Python 或 JavaScript/TypeScript。
- 熟悉常用的 Web 框架（如 FastAPI, Django, Express, React, Vue 等）。
- 具备良好的代码规范和 Git 版本控制能力。
- 熟悉 Linux 环境及常用命令。

## 加分技能
- 了解大模型（LLM）API的调用及 Prompt Engineering。
- 有云原生、Docker、CI/CD 实践经验。
- 具备独立负责小型项目的能力。

## 工作形式
- 远程/弹性工作制，结果导向。

## 薪资区间/职级
- 职级：Lv.1 初级工程师
- 薪资：面议（符合初级工程师市场水平，根据实际能力浮动）。

## 面试流程建议
1. 简历初筛（HR）
2. 在线技术测试/笔试（考察基础代码能力）
3. 技术面试（考察项目经验与技术深度）
4. CEO 终面（考察文化契合度与综合素质）
5. 发放 Offer
