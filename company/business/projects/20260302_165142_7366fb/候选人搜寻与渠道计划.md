# 候选人搜寻与渠道计划

## 1. 目标画像
- **岗位名称**: Python代码工程师
- **职级定位**: 初中级或高级（视具体能力而定，优先寻找具备独立作战能力的复合型人才）
- **核心技能**: 精通Python编程，熟悉主流Web框架（如Django/FastAPI/Flask），具备良好的代码规范和工程化能力。
- **加分项**: 有AI/大模型（LLM）相关项目经验，或全栈开发能力。
- **软素质**: 极强的自驱力，符合“狼性文化”和“实事求是，求是创新”的价值观，具备创业心态，能适应快节奏和高压环境。

## 2. 核心招聘渠道（>=3）

### 2.1 Boss 直聘 (核心主动出击渠道)
- **定位**: 快速覆盖活跃求职者，精准匹配Python后端开发工程师。
- **优势**: 沟通直接，反馈快，适合初中级到高级各种层次的技术人才。
- **预算**: 平台基础套餐或免费权益（如有付费需求另行申请）。
- **策略**: 每日主动开撩50人，重点筛选具备独立作战能力和创业心态的候选人。使用自定义问候语筛选软素质。
- **周期**: 持续进行，预计1-2周内完成首批高质量候选人筛选。
- **每周产出目标**: 每周获取有效简历 >= 15 份，转化初筛合格候选人 >= 5 人。
- **负责人**: Sam HR（暖心侠）

### 2.2 员工内推 (高转化率信任渠道)
- **定位**: 利用现有团队人脉，寻找文化契合度高、能力被验证的人才。
- **优势**: 信任度高，入职后融入快，符合“狼性文化”的筛选门槛，历史履历真实度高。
- **预算**: 内部推荐奖金（视公司现行政策而定，如录用且转正后发放）。
- **策略**: 发布内部推荐悬赏公告，鼓励现有员工推荐前同事或校友。HR主动跟进内推线索优先面试。
- **周期**: 持续进行，视员工推荐频次而定，通常推荐后1周内完成面试流程。
- **每周产出目标**: 每周收集内推简历 >= 2 份，转化面试 >= 1 人。
- **负责人**: Sam HR（暖心侠）及全体员工

### 2.3 GitHub 与 开源社区 / 技术垂直社区 (高质量被动求职者补充渠道)
- **定位**: 寻找技术极客、高级开发人员和具备自驱力的顶尖人才（V2EX/掘金等）。
- **优势**: 代码质量可见，技术实力有保障，能吸引到有特定技术追求（如AI应用）的人才。
- **预算**: 零预算（技术社区免费发帖/GitHub直接联系）。
- **策略**: 
  - 搜索活跃的Python开源项目贡献者，特别是参与过AI/LLM相关项目的人才，通过邮件直接联系。
  - 在V2EX/掘金等发布走心的招聘贴，强调One Man Company的独特模式和高回报。
- **周期**: 长期关注，预计2-3周内建立初步联系。
- **每周产出目标**: 每周触达潜在技术大牛 >= 5 人，收集简历 >= 3 份。
- **负责人**: Sam HR（暖心侠）协同技术团队

## 3. 招聘周期计划
- **第1周 (搜寻与初筛)**: 完善JD，全面铺开招聘渠道（Boss直聘发布、技术社区发帖、启动内推）。完成初步简历收集与初筛（目标 >= 10份有效简历，产出初筛名单 >= 5人）。
- **第2周 (面试与评估)**: 开展首轮技术面试与笔试，筛选出3-5名优质候选人进入终面短名单。
- **第3周 (终面与录用)**: 安排CEO终面，确定最终录用人选，完成背景调查与Offer发放。
- **第4周 (入职与融入)**: 候选人入职跟进，准备入职培训及设备，设定试用期目标。

## 4. 验收与复盘
本计划执行过程中的所有候选人数据需同步记录至《候选人搜寻与初筛台账.md》，以供每周复盘渠道有效性，并作为最终项目验收的证据。
