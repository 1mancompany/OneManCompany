# 代码工程师招聘计划及岗位描述 (JD)

## 1. 岗位需求与技能栈
- **岗位名称**: 初级代码工程师 (Junior Code Engineer)
- **职级**: Lv.1
- **部门**: 技术研发部
- **核心技能栈**: Python, 后端开发, AI 接口集成, 独立解决问题能力
- **文化价值观**: 认同“实事求是，求是创新”以及“狼性文化，适者生存”

## 2. 招聘渠道与预算
- **核心渠道**: Boss 直聘 (Boss Online)
- **预算**: 初级工程师标准薪资，根据面试表现定薪

## 3. 面试流程与评分标准
- **初筛**: 匹配度 > 80%，具备 Python 与 AI 相关经验。
- **面试考核**:
  1. 技术面：现场代码编写、架构设计能力考核。
  2. 价值观面：抗压能力、独立作战能力评估。

## 4. 后续流程
- 搜寻候选人并提取 Top 5 给 CEO 审批。
- 审批后安排面试与 Offer 发放。
