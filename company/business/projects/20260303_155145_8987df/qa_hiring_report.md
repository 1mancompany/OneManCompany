# QA (Test Engineer) Hiring Report

## Job Description (JD)
**Job Title:** Test Engineer (QA) - Lv.1 初级工程师
**Department:** 技术研发部 (Engineering)

**Responsibilities:**
- Design, develop, and execute comprehensive test plans and test cases.
- Perform manual and automated testing of web applications and APIs.
- Identify, record, document thoroughly, and track bugs.
- Collaborate closely with developers to understand features and ensure high-quality delivery.
- Build and maintain automated testing frameworks (e.g., using Python, pytest, Selenium).

**Requirements:**
- Solid understanding of software QA methodologies, tools, and processes.
- Experience in writing clear, concise, and comprehensive test plans and test cases.
- Hands-on experience with both white box and black box testing.
- Proficiency in programming languages such as Python or JavaScript for test automation.
- Strong analytical and problem-solving skills, with a keen eye for detail.
- Excellent communication skills and a team player mindset.
