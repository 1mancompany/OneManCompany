# 《Soul Rift》游戏机制分析报告

根据最新开发的游戏设计文档（GDD），我们刚开发的Roblox游戏《Soul Rift》的核心游戏机制如下：

## 1. 核心玩法循环 (Core Gameplay Loop)

《Soul Rift》是一款动漫风格的快节奏PvE合作Roguelike ARPG（1-4人组队）。游戏融合了《Hades》的爽快战斗与《Dungeon Quest》的装备驱动机制，单局时长在15-20分钟。

### 1.1 单局循环 (Run Loop)
玩家的单局体验遵循经典的Roguelike闯关模式：
1. **大厅准备**：在大厅选择角色和装备。
2. **进入裂隙**：进入程序化生成的地下城（裂隙）。
3. **房间推进**：
   - 进入房间 -> 清理怪物 -> 获得奖励。
   - **核心抉择**：清完战斗房间后，玩家需从3张随机**天赋卡**中选择1张（构建本局的Build）。
   - **路径选择**：选择下一个房间的类型（如战斗房、精英房、商店房、休息房等）。
4. **Boss战**：每层（共8-12个房间）的最后会遭遇当前层Boss。
5. **结算**：击败Boss通关或中途死亡失败，结算获得经验、装备和货币（灵魂碎片）。
6. **返回大厅**：利用战利品进行永久成长。

### 1.2 局外养成循环 (Meta Progression)
游戏包含了跨局的永久成长机制：
- **资源积累**：单局获得的经验值、装备和灵魂碎片。
- **角色成长**：解锁新职业、技能树和被动能力。
- **装备系统**：收集、强化（使用灵魂碎片）、合成装备（3件同稀有度合成更高稀有度）。
- **社交与展示**：参与排行榜、收集赛季通行证外观、在大厅展示成就。

## 2. 核心战斗与角色系统

### 2.1 职业设计
游戏设计了多种职业，初始解锁2个，其余通过游玩解锁：
- **剑士 (Blade Dancer)**：近战，主打快速连击和闪避反击。
- **咒术师 (Soul Caster)**：远程，主打灵魂弹和法术护盾。
- 其他职业包括：拳师（高风险高回报近战）、猎手（高机动远程）、守护者（坦克辅助）和雷灵（高速范围控制）。

### 2.2 技能操作
为适配移动端（Roblox特性），操作设计精简：
- 左侧虚拟摇杆移动。
- 右侧3个主动技能按钮 + 1个终极技能按钮（蓄能释放）。
- 自动普攻（可关闭）+ 独立闪避按钮。
每个职业拥有1个可通过跨局升级强化的专属被动天赋。

## 3. 局内Build构建 (天赋卡系统)

这是单局Roguelike体验的核心：
- **机制**：每个战斗房间后3选1。
- **稀有度**：分为普通、稀有、史诗、传说四档，高稀有度卡牌能改变技能形态或定义Build方向。
- **联动效应 (Synergy)**：同时持有特定天赋卡组合可触发额外强大效果（例如：生命汲取 + 连锁闪电 = 闪电链也能回血）。

## 4. 关卡生成机制 (Procedural Generation)

- **结构**：完整单局包含3层（如幽灵废墟、熔岩神殿、虚空深渊），每层8-12个房间加1个Boss房。
- **生成规则**：保证每层有商店、休息和精英房；Boss前必有休息房；每2-3个房间提供路径分支选择。
- **房间多样性**：从多种预制模板（开阔型、走廊型、多平台型等）中随机选择布局，10%概率出现秘密房间。

## 5. 挑战与难度系统

游戏设置了多级难度曲线（普通、困难、噩梦、深渊、无尽），高难度下敌人倍率增加，并引入特殊规则（如噩梦无休息房，深渊永久死亡无复活），以满足不同水平玩家的需求，填补"超硬核"与"纯休闲"之间的市场空白。
