# 软件工程师（代码工程师）端到端招聘计划

## 1. 岗位需求与级别
- **岗位名称**: 初级工程师 (Junior Software Engineer) - *注：按公司制度所有新员工从Lv.1初级起步*
- **方向**: 全栈 / 后端偏重
- **必备技能**: Python, JavaScript/TypeScript, RESTful API设计, 数据库基础(SQL/NoSQL)
- **加分技能**: DevOps (Docker, CI/CD), 基础架构设计, Go语言经验
- **预算**: 具有竞争力的初创公司薪酬期权包
- **入职时间**: ASAP（越快越好）

## 2. 岗位JD (Job Description)
**岗位职责**:
- 负责公司核心业务系统、API及后台服务的开发与维护。
- 参与产品需求讨论，独立完成模块级的设计、编码与测试。
- 解决开发中的技术难题，持续优化系统性能与稳定性。
- 践行“实事求是，求是创新”及“狼性文化”，在快节奏环境中高效产出。

**任职要求**:
- 计算机相关专业，具备扎实的编程功底。
- 熟练掌握 Python / JavaScript，有实际的后端或全栈开发经验。
- 熟悉常用数据库及缓存技术（如 MySQL, PostgreSQL, Redis）。
- 具备极强的独立作战能力和问题解决能力。

## 3. 招聘渠道与寻访计划
- **主渠道**: BOSS直聘 (Boss Online) - 快速触达活跃候选人。
- **辅助渠道**: 内部员工推荐 (内推) - 确保文化契合度与信任基础。
- **寻访策略**: 重点搜索具有初创公司经验、全栈能力强、能独立负责项目的候选人。

## 4. 简历筛选与初面问题库
**简历筛选标准**:
- 关注独立完成或主导的项目经验。
- 关注技术栈的匹配度（Python/JS）。
- 寻找有“折腾”精神的开源贡献者或技术博客作者。

**初面问题库 (HR & 基础技术考察)**:
1. 请简述你过去最自豪的一个项目，你在其中承担了什么角色？
2. 在遇到完全未知的技术难题时，你通常如何解决？
3. 我们公司推崇“狼性文化”和极强的独立作战能力，你如何理解并在工作中体现？
4. 你的期望薪资和最快入职时间是？

## 5. 技术面试流程与评估表
**面试流程**:
- **一面 (在线/现场)**: 基础技术面（算法基础、语言特性、数据库基础）。
- **二面 (项目实战)**: 架构设计与场景题（如：如何设计一个高并发的API限流系统）。
- **三面 (CEO面)**: 综合素质、文化契合度与终面。

**面试评估表 (评分1-5)**:
- 编程基础与代码规范 ( )
- 系统设计与架构能力 ( )
- 问题解决与学习能力 ( )
- 沟通表达与团队协作 ( )
- 狼性文化与抗压能力 ( )

## 6. offer建议与谈判策略
- **策略**: 强调初创公司的广阔成长空间、核心系统参与感及期权潜在价值。
- **谈判底线**: 依据候选人能力在预算范围内灵活调整，但职级严格锁定为Lv.1，通过未来绩效晋升(连续3季度3.75)获得加薪与职级提升。
- **跟进**: 发放Offer后保持高频沟通，协助解决入职前的顾虑。
